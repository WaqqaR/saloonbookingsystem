# Marketing drafts

The automated marketing agent writes its output here, one folder per run
(`drafts/YYYY-MM-DD/`). Each run produces:

- `social.md` — drafted social posts (X, LinkedIn, Instagram)
- `outreach.md` — drafted cold outreach emails

Everything in here is a **draft for human review**. Review, edit, then post or
send manually. Nothing is published automatically.

See `../BRIEF.md` for the positioning and rules the agent follows.
